Expand description

The most general form of an expression.

As you can see if you read the variant list, in GLSL, an assignment is an expression. This is a bit silly but think of an assignment as a statement first then an expression which evaluates to what the statement âreturnsâ.

An expression is either an assignment or a list (comma) of assignments.
