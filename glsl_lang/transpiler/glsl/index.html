A GLSL450/GLSL460 transpiler that takes a syntax tree and writes it as a plain raw GLSL String.

Â§Foreword

This module exports several functions that just transform a part of a syntax tree into its raw GLSL String representation.

Important note: this module â€“ and actually, any transpiler module â€“ is not responsible in optimizing the syntax tree nor semantically check its validity. This is done in other stages of the compilation process.

In order to achieve that purpose, you could:

  • For each elements in the AST, return a String or Cow<str>.
  • Insert the string representation via a formatter.

The second solution is better because it lets the user handle the memory the way they want: they might just use a dynamic buffer that implements Write or simply pass a &mut String. Itâ€™s up to you.
